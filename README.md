# Make Chess with TCP/IP

### Cài Đặt

**Biên Dịch Sử Dụng Makefile:**
   ```bash
   make all
   ```

#### Sử dụng Makefile

1. **Chạy Server:**
   ```bash
   make run_server
   ```

2. **Chạy Client:**
   Mở một terminal mới cho mỗi client và chạy:
   ```bash
   make run_client
   ```

### Dọn Dẹp
```bash
make clean
```
